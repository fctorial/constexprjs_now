## ConstexprJS Now

This is a clone of [Jekyll Now](https://github.com/barryclark/jekyll-now) theme, built with
[ConstexprJS](https://github.com/fctorial/ConstexprJS).

### Demo

A demo of this theme has been deployed [here](https://fctorial.github.io/raw/constexprjs_now).

### Development

Use this command to serve the site:

    python -m http.server 3000

The site will be served at `https://localhost:3000`.

### Building

Use `./build.sh` script to compile the website. The final website will be placed in the
directory `../constexprjs_now_deployment`.
